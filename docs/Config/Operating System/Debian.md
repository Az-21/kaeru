# Debian
## Initial Setup
```bash
sudo apt update
sudo apt upgrade

# Dependencies
sudo apt install curl wget git

# Starship.rs
curl -sS https://starship.rs/install.sh | sh

# Python -> uv package manager
wget -qO- https://astral.sh/uv/install.sh | sh
```
